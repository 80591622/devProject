# React 内容管理系统

一个基于React的现代化内容管理系统，用于管理和发布文章。

## 技术栈

- **前端框架**: React 18
- **状态管理**: Redux Toolkit
- **路由管理**: React Router 7
- **UI组件库**: Ant Design 5
- **富文本编辑器**: Toast UI Editor
- **图表库**: ECharts
- **HTTP请求**: Axios
- **样式预处理器**: Sass
- **构建工具**: Create React App + Craco

## 功能特性

### 核心功能

- ✅ 用户登录/退出
- ✅ 文章列表展示
- ✅ 文章发布与编辑
- ✅ 文章删除功能
- ✅ 文章筛选（按状态、频道、日期）
- ✅ 封面图片上传
- ✅ 富文本编辑
- ✅ 频道管理
- ✅ 数据可视化图表

### 技术特性

- 📱 响应式布局设计
- 🔒 路由权限控制
- 🎨 现代化UI设计
- 📊 数据统计图表
- 📝 富文本编辑功能
- 🔄 异步数据处理

## 项目结构

```
src/
├── apis/          # API接口定义
├── assets/        # 静态资源
├── components/    # 通用组件
├── hooks/         # 自定义Hook
├── pages/         # 页面组件
│   ├── Article/   # 文章管理
│   ├── Home/      # 首页/仪表盘
│   ├── Layout/    # 布局组件
│   ├── Login/     # 登录页面
│   └── Publish/   # 文章发布/编辑
├── router/        # 路由配置
├── store/         # Redux状态管理
├── utils/         # 工具函数
├── App.js         # 应用入口组件
└── index.js       # 项目入口文件
```

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:3000` 启动

### 构建生产版本

```bash
npm run build
```

### 运行测试

```bash
npm run test
```

## 使用说明

### 登录

使用以下测试账号登录系统：

- **用户名**: 13800000002
- **验证码**: 246810

### 文章管理

1. **文章列表**: 首页展示所有文章，支持筛选和分页
2. **发布文章**: 点击"发布文章"按钮，填写标题、选择频道、上传封面、编辑内容
3. **编辑文章**: 在文章列表点击编辑图标，修改文章内容
4. **删除文章**: 在文章列表点击删除图标，确认后删除文章

### 富文本编辑器

系统使用Toast UI Editor提供富文本编辑功能，支持：

- 文本格式化
- 列表、表格、链接
- 图片上传
- 代码块

## 开发指南

### API接口

项目使用模拟API进行开发，接口地址配置在 `src/utils/request.js` 中。

### 自定义Hook

- `useChannel`: 获取频道列表

### 组件开发

遵循React组件开发最佳实践：

- 组件化设计
- 合理使用Hooks
- 状态管理清晰
- 代码注释完整
