# React 学习与演示项目集合

一个包含多个React相关学习和演示项目的集合，涵盖了从基础到高级的各种React技术和最佳实践。

## 项目列表

### 1. react-jike - React内容管理系统

一个基于React的现代化内容管理系统，用于管理和发布文章。

**主要功能**：

- 用户登录/退出
- 文章管理（发布、编辑、删除）
- 频道管理
- 数据可视化图表

**技术栈**：React 18, Redux Toolkit, React Router 7, Ant Design 5, Toast UI Editor

**启动方式**：

```bash
npm install
npm run dev
```

### 2. react-bill-test - React账单管理应用

一个基于React的现代化账单管理系统，用于记录和统计个人收支情况。

**主要功能**：

- 收支记录添加
- 账单分类管理
- 月度/年度账单查询
- 收支数据统计

**技术栈**：React 18, Redux Toolkit, Ant Design Mobile 5, JSON Server

**启动方式**：

```bash
npm install
npm run dev  # 同时启动前端和后端服务
```

### 3. react-pro - React特性演示项目

一个专注于演示React核心特性和最佳实践的学习项目。

**主要内容**：

- 类组件相关（生命周期、通信）
- React Hooks完整演示
- Zustand状态管理

**启动方式**：

```bash
npm install
npm start
```

### 4. react-router-pro - React Router演示项目

一个专注于演示React Router核心功能和最佳实践的学习项目。

**主要内容**：

- 基础路由配置
- 嵌套路由实现
- 路由参数传递
- 编程式导航
- 路由守卫

**启动方式**：

```bash
npm install
npm start
```

### 5. redux-meituan - Redux美团外卖演示项目

一个基于React和Redux的美团外卖应用演示项目。

**主要功能**：

- 商品分类展示
- 商品列表浏览
- 购物车功能
- 商品数量管理

**技术栈**：React, Redux Toolkit, Sass, JSON Server

**启动方式**：

```bash
npm install
npm run serve  # 启动后端服务
npm start      # 启动前端服务
```

### 6. react-ts-tabs - React TypeScript Tabs演示项目

一个基于React + TypeScript + Vite的Tabs组件演示项目。

**主要功能**：

- Tabs标签页切换
- 数据列表展示
- 详情页查看
- 异步数据加载

**技术栈**：React 18, TypeScript, Vite, Axios

**启动方式**：

```bash
npm install
npm run dev
```

## 学习资源

每个项目都包含了详细的README文件，介绍了项目的功能、技术栈和启动方式。建议按照以下顺序学习：

1. react-pro - 掌握React基础特性
2. react-router-pro - 学习React Router
3. redux-meituan - 理解Redux状态管理
4. react-bill-test - 学习完整应用开发
5. react-jike - 掌握企业级应用开发
6. react-ts-tabs - 学习TypeScript + Vite
